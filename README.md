# bridge
selendra bridge with rust

## smart-contract
- go to smart contract directory
```sh
cd smart-contracts
```

- compile smart contract
```sh
truffle compile
```
- run ganache for testing

```sh
./scripts/start_ganache.sh
```

- run test all
```sh
truffle test
```

- run test witg specific file
```sh
truffle test <test-file>
```
